# Instruções para Screenshots

Este diretório contém screenshots do sistema Argo Mobile para complementar a documentação do projeto.

## Páginas a Serem Capturadas

Para uma documentação completa, capture screenshots das seguintes páginas:

### Site Institucional
1. **homepage.png** - Página inicial (Home)
2. **services.png** - Página de serviços
3. **portfolio.png** - Página de portfólio
4. **about.png** - Página sobre a empresa
5. **contact.png** - Página de contato
6. **login.png** - Página de login

### Sistema Interno
7. **dashboard.png** - Dashboard administrativo
8. **projects.png** - Lista de projetos
9. **project-detail.png** - Página de detalhes do projeto
10. **tasks.png** - Página de gerenciamento de tarefas
11. **team.png** - Página de gerenciamento da equipe
12. **profile.png** - Página de perfil do usuário

## Instruções de Captura

1. Utilize a resolução 1920x1080 para todas as capturas para manter consistência
2. Capture a tela inteira, incluindo o cabeçalho e rodapé quando aplicável
3. Faça login com diferentes tipos de usuário para capturar as diferentes interfaces:
   - Administrador (acesso completo)
   - Gerente (gerenciamento de projetos)
   - Programador (visualização de tarefas)
   - Designer (visualização de tarefas)
4. Nomeie os arquivos exatamente como listado acima para manter a consistência com o README

## Inclusão no README

Após capturar as imagens, substitua os placeholders no arquivo README.md pelos caminhos reais para as imagens:

```markdown
![Página Inicial](screenshots/homepage.png)
```

Certifique-se de que todas as imagens estão no formato PNG e foram otimizadas para web.
